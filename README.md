#  Password Strength Checker

This is a Python program that checks the strength of a password based on different criteria.

##  Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Classifies password as Weak, Medium, or Strong

##  How to Run
1. Install Python
2. Run: python password_checker.py
3. Enter a password

##  Learning Outcome
- Input validation
- Regular expressions
- Basic cybersecurity practices
